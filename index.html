Embedding nextstrain in an existing website

A simple example of how to embed a nextstrain visualisation inside an already existing website using iFrames. This relies on auspice PR #884 which is deployed as a review app here.

We'll set a "simple" iframe to nextstrain.org/zika with URL queries which close the sidebar, hide the header+footer, and only show the tree. Note that the iframe size is hardcoded to 600x800 in this example.
